# 함수

## 특징
1. 함수의 목적 : 중복을 줄이기 위함
2. 한 번에 한 작업에 집중
3. 읽기 쉽고 어떤 동작인지 알 수 있게 네이밍
  ```jsx
  showError() //에러를 보여줌
  getName() // 이름을 얻어옴
  createUserData() //유저데이터 생성
  checkLogin() // 로그인 여부 체크
  ```
<br>

## 매개변수 없는 함수
  ```jsx
  function showError() {
    alert('에레가 발생했습니다. 다시 시도해주세요.')
  }
  
  showError();
  ```
<br>

## 매개변수 있는 함수
  ```jsx
  function sayHello(name) {
    const msg = `Hello, ${name}`; 
    console.log(msg);
  }
  
  sayHello('Yueun'); //"Hello, Yueun"
  
  function sayHello(name) {
    let msg = 'Hello'; //msg는 함수 밖에서는 못 씀
    if(name){
      msg += ', ' + name;
    }
    console.log(msg);
  }
  
  sayHello(); // "Hello"
  sayHello('Yueun') // "Hello, Yueun"
  
  ```
<br>

## 함수 선언문 
: 어디서든 호출 가능
  ```jsx
  sayHello(); // 함수식 위에 있어도 상관없다!
  
  function sayHello(){
    console.log('Hello');
  }
  ```
<br>

## 함수 표현식 
: 코드에 도달하면 생성(사용 전에 선언 필요)
  ```jsx
  ...
  ...
  ```
